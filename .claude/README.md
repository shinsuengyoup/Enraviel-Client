# Claude Code 프로젝트 설정

이 디렉토리는 Claude Code를 위한 프로젝트 특화 설정을 포함합니다.

## 파일 목록

- **CODING_GUIDELINES.md** - 프로젝트 코딩 규칙 및 스타일 가이드
- **custom_instructions.md** - Claude Code에 제공되는 커스텀 지시사항

## 사용 방법

Claude Code는 이 디렉토리의 파일들을 자동으로 참조하여 프로젝트의 코딩 스타일과 규칙을 따릅니다.

코드 작성 시 Claude Code에게 "코딩 가이드라인을 따라서 작성해줘"라고 요청하면 CODING_GUIDELINES.md를 참고하여 코드를 생성합니다.
